The One with the Dash - PWA (Original Endless Runner)

Files included:
- index.html, style.css, game.js
- manifest.json, service-worker.js
- bg_loop.wav (royalty-free loop)
- icon-*.png (72-512)

How to run:
1) Unzip and serve with a static server: `python -m http.server 8080`
2) Open http://localhost:8080
3) Click "Play Music" to enable audio (autoplay is blocked).
4) Install via browser menu to add to home screen.

Notes:
- This is an original game inspired by lane-based endless runners. It avoids copyrighted names, logos, or characters.
